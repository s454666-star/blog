# Folder Video Android APK

Native Android WebView wrapper for the local `blog` Folder Video app.

- Default URL: `http://10.0.0.8/folder-video-app`
- Package: `monster.mystar.foldervideo`
- Min SDK: 23
- Target SDK: 35

The Laravel app remains the backend and UI source. The APK exists so Android can install and launch it like a normal app.
